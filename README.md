# Description
Continuation of my [Engineer's thesis](https://github.com/RobertMut/HomeStation) to manage and collect data in home.

## Technologies used
### API and Front-end side:
- Docker
- Angular
- .Net8 + ASP.Net
- Entity framework
- MQTTNet
### Device side:
- C/C++
- Bosch's BME280 driver
- FreeRTOS with LwIP
- PlatformIO
- ESP-IDF
